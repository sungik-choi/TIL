#### fetch API

```JavaScript

```

|  종류   | 동기            | 비동기                  |
| :---: | ------------- | -------------------- |
|   1   | fetch()       | 서버 요청                |
|   2   | then()        | 콜백등록(1)              |
|   3   | then()        | 콜백등록(2)              |
|   4   | console.log() | -                    |
|   5   | -             | fecth 서버 응답          |
|   6   | -             | 콜백반환(1)              |
|   7   | -             | 콜백반환(1)의 값으로 콜백반환(2) |
